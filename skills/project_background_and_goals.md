# 项目背景与目标

## 项目概述

**项目名称**：BuggyProject  
**项目类型**：自主寻线小车  
**核心技术**：电感感应 + 闭环控制  

---

## 硬件平台

### 核心芯片
- **型号**：Microchip PIC16F18326
- **开发工具**：MPLab X + MCC（MPLAB Code Configurator）
- **编程语言**：C 语言

### 硬件栈
- 传感器子系统：电感感应电路，输出模拟信号
- 控制子系统：PIC16F18326 单片机
- 执行子系统：H 桥电路驱动电机
- 接口：ADC 模拟输入 + PWM 数字输出

---

## 系统架构

### 执行逻辑
采用 **时间触发调度（Time Triggered Scheduling）**

- **中断周期**：通常为 50ms
- **工作模式**：周期性任务执行
- **控制方式**：基于中断的实时响应

### 数据流向
```
传感器输入 → ADC 读取 → 控制算法处理 → PWM 输出 → 电机驱动
```

---

## 我的角色定位

**核心职责**：Control Subsystem（控制系统）开发  

**工作内容**：
1. **接口配置**
   - 编写代码读取传感器模拟信号（ADC）
   - 将模拟信号转换为数字控制量

2. **算法设计**
   - 实现 P / PI / PID 控制算法
   - 调试参数以确保性能最优
   - 实时调整转向角度和电机速度

3. **输出驱动**
   - 将计算结果映射为 PWM 占空比
   - 通过 H 桥电路控制电机运行

---

## 项目目标

### 1. 闭环稳定性
实现转向角度与传感器输入之间的精确闭环控制，确保系统响应准确。

### 2. 自主性
小车能够根据感应电压的变化，自主且平滑地修正行驶路径，无需人工干预。

### 3. 性能优化
通过优化 PID 参数，实现：
- 最快的巡线速度
- 最小的震荡幅度
- 保持系统稳定性

---

## 技术要点

### 控制算法
- **P 控制**：比例控制，快速响应
- **PI 控制**：比例积分控制，消除稳态误差
- **PID 控制**：比例积分微分控制，综合优化

### 关键指标
- 采样精度：10 位 ADC
- 控制频率：20Hz（50ms 周期）
- 响应时间：< 周期内完成
- 稳态误差：< 1%
